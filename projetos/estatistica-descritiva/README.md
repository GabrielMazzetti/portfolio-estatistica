# Estatística Descritiva

Repositório com exercícios, notebooks e scripts sobre estatística descritiva: medidas de tendência central, dispersão, visualizações e sumários.

Como usar
- Coloque aqui notebooks Jupyter (`.ipynb`), scripts Python/R e arquivos de dados.
- Execute os notebooks no seu ambiente local (ex.: `jupyter lab`).

Sugestões de arquivos
- `notebook-exemplo.ipynb` — exemplo com importação de dados e gráficos
- `dados/` — pasta para CSVs e datasets
