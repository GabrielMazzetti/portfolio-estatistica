# Modelagem

Projetos de modelagem estatística e machine learning: preparação de dados, pipelines, validação cruzada e avaliação de modelos.

Como usar
- Inclua notebooks com experimentos e um arquivo `requirements.txt` quando necessário.
- Documente como reproduzir os resultados no próprio notebook.

Sugestões de arquivos
- `experimento.ipynb`
- `requirements.txt`
