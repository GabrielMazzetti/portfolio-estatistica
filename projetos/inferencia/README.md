# Inferência Estatística

Material sobre inferência estatística: testes de hipótese, intervalos de confiança, regressão e modelagem probabilística.

Como usar
- Adicione notebooks com exemplos e scripts para reproduzir análises.
- Inclua dados e instruções de execução nos notebooks.

Sugestões de arquivos
- `notebook-teste-hipotese.ipynb`
- `scripts/` — scripts para rodar análises
